# Brand assets

Artwork for the [home-assistant/brands](https://github.com/home-assistant/brands)
submission. Copy `custom_integrations/sensoredlife/` into a fork of that repo
under the same path and open a PR.

| File | Size | Notes |
| --- | --- | --- |
| `icon.png` / `icon@2x.png` | 256² / 512² | Square "halo" icon, dark house |
| `dark_icon.png` / `dark_icon@2x.png` | 256² / 512² | Light house for dark themes |
| `logo.png` / `logo@2x.png` | 465×256 / 930×512 | Faithful wide logo, dark house |
| `dark_logo.png` / `dark_logo@2x.png` | 465×256 / 930×512 | Light house for dark themes |

Brand colors: dots `#23A0F4`, house `#0F0F0F` (light variant `#F4F4F4`).
All transparent and trimmed (content touches all edges). The square icon
re-arranges the seven dots into a ring around the house because the brands
icon must be naturally square (1:1); the wide arch is preserved in `logo.*`.
